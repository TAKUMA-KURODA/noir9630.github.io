## 3. スタイル定義の特性
タイポグラフィ: Google Fontsの「M PLUS Rounded 1c」を優先フォントとして指定している。  
レイアウト: FlexboxおよびGridレイアウトを採用し、要素の動的な配置を行っている。  

視覚効果: スティッキーヘッダー（上部固定）およびカード要素へのホバーアニメーションが実装されている。
